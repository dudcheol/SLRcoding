# 한이음 프로보노 프로젝트
##  모바일 캠퍼스라이프 커뮤니티 앱

__박영철, 김연준, 이정찬, 최민철__

### 0. 개요
------------------------------
기존의 모바일에서 제공되는 학교생활 정보 애플리케이션은 단방향으로 정보만을 제공하기 때문에 실용성에 한계가 있다. 
본 프로젝트에서는 대학교에서 많은 학생들이 실제 학교생활에서 필요로 하는 물품 거래와 인적 교류 커뮤니티를 구축하고,
학생들이 상호 필요로 하는 물품이나 친구찾기를 스마트폰에서 실시간으로 등록, 검색, 수정, 삭제할 수 있으며, 
거래를 원하는 학생이 존재하면 직거래 혹은 친구 찾기를 매칭시켜주는 쌍방향 맞춤형 플랫폼을 구축한다.

### 1. 제작 목적
------------------------------
* 프로젝트 소개
본 프로젝트는 학교 웹 사이트나 일반적으로 모바일에서 제공되는 학교 생활 정보 제공 애플리케이션과 차별화하여 모바일을 기반으로 학생들이 일상에서 필요로 하는 물품 교류 및 친구 찾기 커뮤니티, 두 가지 커뮤니티에 초점을 두고 구축하는데 목표를 둔다.
- 물품 거래 : 책자, 컴퓨터, 생활용품, 가구, 가전, 소품등
- 친구 교류 : 함께 기숙할 친구찾기, 스포츠, 게임, 동호회, 스터디 그룹 친구 찾기
